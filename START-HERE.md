# Braintopia Pillar 1 — Getting Started

This is a complete Next.js web app built exactly to the PLAN.md spec.

## 1. First Time Setup (on your machine)

```bash
cd braintopia-pillar1-education

# Install dependencies
npm install

# Copy the logo so the app can use it
mkdir -p public
cp assets/braintopia-logo.png public/braintopia-logo.png
```

## 2. Run the App

```bash
npm run dev
```

Open http://localhost:3000
